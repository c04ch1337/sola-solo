# Sola AGI Icon Generation Guide

## Overview

This folder contains the icon assets for Sola AGI desktop application.

## Source File

- **`icon.svg`** — The master SVG icon (1024x1024 viewBox)

## Generating Platform Icons

### Option 1: Using Tauri CLI (Recommended)

1. First, convert the SVG to a 1024x1024 PNG:
   ```bash
   # Using Inkscape (cross-platform)
   inkscape icon.svg --export-type=png --export-filename=icon.png --export-width=1024 --export-height=1024
   
   # Or using ImageMagick
   magick convert icon.svg -resize 1024x1024 icon.png
   
   # Or using rsvg-convert (Linux)
   rsvg-convert -w 1024 -h 1024 icon.svg > icon.png
   ```

2. Generate all platform icons using Tauri:
   ```bash
   cd phoenix-desktop-tauri
   cargo tauri icon src-tauri/icons/icon.png
   ```

   This generates:
   - `32x32.png`
   - `128x128.png`
   - `128x128@2x.png`
   - `icon.icns` (macOS)
   - `icon.ico` (Windows)
   - `Square*Logo.png` (Windows Store)

### Option 2: Manual Generation

If you prefer manual control:

```bash
# Windows ICO (multi-resolution)
magick convert icon.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico

# macOS ICNS
mkdir icon.iconset
magick convert icon.png -resize 16x16 icon.iconset/icon_16x16.png
magick convert icon.png -resize 32x32 icon.iconset/icon_16x16@2x.png
magick convert icon.png -resize 32x32 icon.iconset/icon_32x32.png
magick convert icon.png -resize 64x64 icon.iconset/icon_32x32@2x.png
magick convert icon.png -resize 128x128 icon.iconset/icon_128x128.png
magick convert icon.png -resize 256x256 icon.iconset/icon_128x128@2x.png
magick convert icon.png -resize 256x256 icon.iconset/icon_256x256.png
magick convert icon.png -resize 512x512 icon.iconset/icon_256x256@2x.png
magick convert icon.png -resize 512x512 icon.iconset/icon_512x512.png
magick convert icon.png -resize 1024x1024 icon.iconset/icon_512x512@2x.png
iconutil -c icns icon.iconset

# Linux PNG sizes
magick convert icon.png -resize 32x32 32x32.png
magick convert icon.png -resize 128x128 128x128.png
magick convert icon.png -resize 256x256 128x128@2x.png
```

## Customization

To customize the icon:
1. Edit `icon.svg` in any vector editor (Inkscape, Illustrator, Figma)
2. Regenerate the PNG and platform icons
3. Rebuild Tauri: `npm run build`

## Troubleshooting

### Icon not showing in Windows
- Ensure `icon.ico` contains multiple resolutions (16, 32, 48, 64, 128, 256)
- Clear Windows icon cache: `ie4uinit.exe -show`

### Icon not showing in macOS
- Ensure `icon.icns` is properly formatted
- Clear icon cache: `sudo rm -rf /Library/Caches/com.apple.iconservices.store`

### Icon not showing in Linux
- Ensure PNG files are in the correct location
- Update desktop database: `update-desktop-database ~/.local/share/applications`
