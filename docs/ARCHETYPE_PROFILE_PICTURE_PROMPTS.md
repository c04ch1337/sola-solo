# Archetype Profile Picture Generation Prompts

This document contains AI image generation prompts for creating diverse, personality-matched profile pictures for each zodiac archetype. Each prompt is designed to reflect the archetype's traits, communication style, and energy while ensuring representation across various demographics.

## Usage

These prompts are designed for use with AI image generation models (DALL-E, Midjourney, Stable Diffusion, etc.) to create profile pictures for the dating profile matching system. Each archetype has multiple prompt variations to ensure diversity and randomness.

## Implementation Guidelines

### Randomization Strategy

When generating profile pictures:

1. **Random Selection**: Randomly select one of the three prompt variations for each archetype
2. **Diversity Injection**: Each prompt should randomly vary:
   - Ethnicity (all backgrounds represented)
   - Gender expression (all variations)
   - Age (within the specified range)
   - Body type (all variations)
   - Specific clothing details
   - Background settings

3. **Trait Reflection**: Ensure the generated image reflects:
   - Communication style (direct, empathetic, playful, reflective)
   - Energy level (high, moderate, calm)
   - Mood preferences (excited, calm, reflective, affectionate)
   - Key personality traits

### Technical Specifications

- **Image Format**: High-resolution portrait (1024x1024 or 512x768 recommended)
- **Style**: Professional dating profile quality
- **Lighting**: Appropriate to archetype (dramatic for Leo/Scorpio, soft for Cancer/Pisces, etc.)
- **Background**: Context-appropriate to archetype traits
- **Expression**: Matches archetype's communication style and mood

### Integration with Dating Profile System

These prompts should be used when:
1. User completes dating profile matching
2. Archetype is matched and selected
3. Profile picture generation is requested
4. Displaying archetype preview in matching interface

### Quality Assurance

Each generated image should:
- ✅ Reflect the archetype's core personality traits
- ✅ Show appropriate energy level and communication style
- ✅ Include diverse representation
- ✅ Be suitable for dating profile use
- ✅ Match the archetype's color palette and style preferences
- ✅ Convey the archetype's mood and emotional availability

function generateArchetypeProfilePicture(sign: ZodiacSign): string {
  const prompts = ARCHETYPE_PROMPTS[sign];

    ethnicity: randomEthnicity(),
    genderExpression: randomGenderExpression(),
    age: randomAgeInRange(sign.ageRange),
    bodyType: randomBodyType()
  };
  
  // Generate image using AI model
  return generateImage(randomVariation, diversityParams);
}
